## 📋 IP 规划框架

### 一、账号定位

#### 1. 账号名称设计

**命名公式**:
```
公式 A: 昵称 + 领域 (如：小王说科技)
公式 B: 昵称 + 功能 (如：李姐帮你挑)
公式 C: 领域 + 人设 (如：程序猿阿强)
公式 D: 特色 + 昵称 (如：会做饭的程序员)
```

**命名原则**:
- ✅ 易记易搜 (2-5 字为佳)
- ✅ 体现领域
- ✅ 有个人特色
- ✅ 避免生僻字
- ❌ 不要纯英文
- ❌ 不要特殊符号

#### 2. 个人简介设计

**简介公式**:
```
【身份标签】+【专业背书】+【内容价值】+【行动引导】

示例:
95 后产品经理 | 5 年互联网经验
分享职场干货和成长心得
关注我，一起变成更好的自己
📮 合作：wechat_xxx
```

**简介要素**:
- 身份标签 (你是谁)
- 专业背书 (为什么信你)
- 内容价值 (关注你能得到什么)
- 行动引导 (私信/合作/社群)

#### 3. 头像设计

**头像类型**:
- 真人头像 (信任度高，推荐)
- 卡通头像 (亲和力强)
- 品牌 Logo (专业感强)

**头像原则**:
- ✅ 清晰高清
- ✅ 主体突出
- ✅ 色彩鲜明
- ✅ 与领域匹配

```
标题：数字 + 痛点/利益点 + 情绪词
封面：高清 + 文字 + 对比
内容：开头共鸣 + 中间干货 + 结尾互动
标签：5-10 个精准标签
```

**抖音爆款公式**:
```

结尾：引导 (点赞/评论/关注)
BGM：热门音乐
```

### 三、IP 延展性路线

#### 阶段一：起号期 (0-1 万粉)

**目标**: 建立人设，积累基础粉丝
**周期**: 1-3 个月
**重点**:
- 确定内容方向
- 保持稳定更新
- 测试爆款内容
- 建立粉丝群

#### 阶段二：增长期 (1-10 万粉)

**目标**: 扩大影响，建立 IP 认知
**周期**: 3-6 个月
**重点**:
- 复制爆款内容
- 跨平台分发
- 品牌合作试探
- 私域流量沉淀

#### 阶段三：变现期 (10 万粉+)

**目标**: 多元化变现，IP 商业化
**周期**: 6-12 个月
**重点**:
- 品牌广告
- 直播带货
- 知识付费
- 自有产品

#### 阶段四：矩阵期 (多账号)

**目标**: IP 矩阵，风险分散
**周期**: 12 个月+
**重点**:
- 子账号孵化
- 团队化运营
- 供应链建设
- 品牌化运营

**封面设计**:

- 突出个人特色

**内容结构**:
```
1. 自我介绍 (你是谁)
2. 为什么做这个账号 (初心)
3. 会分享什么内容 (价值)
4. 互动引导 (关注我)
```

#### 抖音第一篇

**内容类型**: 口播/展示

**脚本结构**:
```
0-3 秒：钩子 (我是 XX，做了 XX 年 XX)
3-15 秒：内容 (分享一个干货/展示一个技能)
15-30 秒：引导 (关注我，分享更多 XX)
```
