## 📋 DNS 云朵说明

### ❌ 为什么不能用橙色云朵（代理）

| 问题 | 说明 |
|------|------|
| 🚫 证书冲突 | Cloudflare 会提供自己的 SSL 证书，与 GitHub Pages 的证书冲突 |
| 🚫 验证失败 | GitHub 无法验证域名所有权 |
| 🚫 HTTPS 错误 | 可能出现重定向循环或证书警告 |

### ✅ 必须用灰色云朵（DNS Only）

| 优势 | 说明 |
|------|------|
| ✅ 直接解析 | DNS 直接指向 GitHub Pages IP |
| ✅ 自动证书 | GitHub 自动管理 HTTPS 证书 |
| ✅ 无冲突 | 不会有 SSL 证书问题 |

## 🎵 添加背景音乐

```bash
# 下载或准备一个轻音乐 MP3 文件
# 复制到 public 目录
cp your-music.mp3 docs/public/bgm.mp3

# 提交推送
git add docs/public/bgm.mp3
git commit -m "add: 背景音乐"
git push origin main
```

**推荐音乐类型**：
- 轻音乐 / 钢琴曲
- Lo-fi Hip Hop
- 环境音乐
- 白噪音

**注意**：音乐文件不要太大（建议 < 5MB）
