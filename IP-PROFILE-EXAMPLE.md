### 四、IP 延展路线

#### 阶段一：起号期 (0-1 万粉)
**周期**: 1-3 个月
**目标**: 建立人设，积累基础粉丝

### 五、设计元素规范

#### 视觉风格

**主色调**:
```
蓝色系：#2E5CFF (专业/信任)
辅助色：#F5F5F5 (背景)
强调色：#FF6B6B (重点/CTA)
```

**字体**:
```
标题：思源黑体 Bold
正文：思源黑体 Regular
英文：Montserrat
```

**封面模板**:
```
布局：左文右图 / 上文下图
文字：不超过 15 字
图片：高清真人/场景
Logo: 右下角固定位置
```

#### 语言风格

**语气**: 专业但不失亲和
**人称**: 我/你 (拉近距离)
**句式**: 短句为主，易读易懂
**表情**: 适量 emoji (每段 1-2 个)

**示例**:
```
❌ 过于正式：
"在职场环境中，有效沟通是至关重要的能力。"

✅ 推荐风格：
"职场沟通，其实就这 3 个技巧！"
```
