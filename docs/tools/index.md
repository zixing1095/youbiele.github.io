## 📐 建站指南

如果你也想搭建个人博客，可以参考以下步骤：

### 快速起步

```bash
# 1. 安装 Node.js（官网下载）

# 2. 初始化项目
mkdir my-blog && cd my-blog
npm init -y
npm install -D vitepress

# 3. 创建首页
echo '# 我的博客\n\nHello World!' > index.md

# 4. 本地预览
npx vitepress dev
```

### 部署到 GitHub Pages

1. 创建仓库 `你的用户名.github.io`
2. 添加 `.github/workflows/deploy.yml`
3. git push → 自动部署
