## 🤔 为什么词根词缀有效？

### 传统背单词 vs 词根记忆

| 方式 | 示例 | 记忆难度 |
|------|------|---------|
| **死记硬背** | vocabulary = v-o-c-a-b-u-l-a-r-y | ❌ 容易忘 |
| **谐音法** | vocabulary = "我怕背累" | ⚠️ 勉强记住 |
| **词根词缀** | voc=说, abul=能够, ary=名词后缀 | ✅ 理解记忆 |

**核心优势**：
- 🎯 理解词义，不是死记
- 🔗 一个词根，记住一串
- 📈 词汇量呈指数增长
- 🔍 猜词能力大幅提升

## 💡 阿野的词根记忆技巧

### 1. 联想记忆法

**例子：port = 拿**

> "Port 是港口，港口是**拿**货的地方"
> import = im（进来）+port = 进口
> export = ex（出去）+port = 出口
> portable = port（拿）+able（能）= 能拿的 → 便携式

### 2. 图像记忆法

把词根画成小图案，贴在常见的地方：
- 📌 port = 👜 背包
- 📌 spect = 👀 眼睛
- 📌 duct = ➡️ 箭头

### 3. 故事记忆法

**例子：proceed = 继续**

> "Pro（往前）+ceed（走）= 往前走 = 继续前进"

### 4. 闪卡复习法

使用 **Anki** 制作词根闪卡：
- 正面：词根
- 背面：含义 + 3个例子
