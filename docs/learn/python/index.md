# Python 学习

> 从 0 到 1，3 个月入门实战

## 📚 学习路线

### 第一阶段：基础入门（第 1-4 周）
- Python 环境搭建
- 基础语法（变量、数据类型、运算符）
- 流程控制（条件、循环）
- 函数与模块
- 文件操作

### 第二阶段：进阶提升（第 5-8 周）
- 面向对象编程
- 异常处理
- 常用标准库
- 第三方库使用
- 实战小项目

### 第三阶段：实战应用（第 9-12 周）
- 数据分析（pandas、numpy）
- 网络爬虫（requests、BeautifulSoup）
- 自动化脚本
- 个人项目开发
