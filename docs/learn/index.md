## 📋 学习方法论

### 学习原则
1. **每天至少 30 分钟** — 坚持比强度重要
2. **输出倒逼输入** — 学完就写笔记
3. **在应用中学习** — 先做再完善
4. **建立知识体系** — 用笔记串联知识点

### 时间管理
| 时间段 | 活动 | 时长 |
|:--:|------|:--:|
| 早上通勤 | 英语听力 | 20 分钟 |
| 午休 | Python 练习 | 30 分钟 |
| 晚间 | 深度学习 + 笔记 | 1 小时 |

**总计：每天至少 1.5 小时**
