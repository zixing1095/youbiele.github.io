¶ Day3 我终于搞懂变量了

姐妹们！我今天可太激动了！😭

之前看代码一堆 $、% 啥的，完全不知道在干嘛。今天终于搞懂了一个最基础的概念——变量。

¶ 变量是什么？

说人话就是：找个盒子，把数据存进去，下次要用直接拿盒子就行。

比如我们记笔记：

  • 名字 = "阿野" → 盒子里写着"阿野"
  • 年龄 = 25 → 盒子里写着 25
  • 工资 = 8000 → 盒子里写着 8000

¶ 为什么要用变量？

不用变量：

python
print("阿野今年25岁，月薪8000")

用变量：

python
name = "阿野"
age = 25
salary = 8000
print(f"{name}今年{age}岁，月薪{salary}")

区别在哪？下次我要改年龄，不用满篇去找25，直接改 age = 26 就行！

¶ 我踩的坑

  1. 忘记加引号

    • name = 阿野 → ❌ 报错！
    • name = "阿野" → ✅ 才对！

  2. 字符串和整数分不清

    • age = "25" → 这是字符串，不是数字！
    • age = 25 → 这才是整数！

  3. True要大写

    • 我之前一直写 true，报错10分钟 🤦‍♂️

¶ 今日代码

python
# 我的第一个变量程序
name = "阿野"
age = 25
is_learning = True

print(f"我是{name}，今年{age}岁")
print(f"我正在学Python：{is_learning}")

我是阿野，今年25岁
我正在学Python：True
