¶ Day4 运算符也太好玩了吧

姐妹们！今天学的运算符也太实用了吧！🎉

以前总觉得编程很高深，今天发现运算符就是小学数学！瞬间觉得自己又行了！

¶ 什么是运算符？

就是 + - * / 这些符号，Python里用它来计算。

¶ 整除和除法的区别

python
print(5 / 2)    # 2.5 - 小数除法
print(5 // 2)   # 2 - 整数除法，只取整数部分

¶ 取余有什么用？

判断奇偶数！

python
print(5 % 2)    # 1 - 奇数
print(4 % 2)    # 0 - 偶数

判断能否整除！

python
# 一个月工作22天，今天是第几天？

¶ 今日代码 - 工资计算器

python
# 阿野的工资计算器
base_salary = 8000  # 基本工资
overtime_hours = 10  # 加班时长
overtime_rate = 50   # 加班费每小时

# 计算加班费
overtime_pay = overtime_hours * overtime_rate

# 计算总工资
total = base_salary + overtime_pay

# 计算税后工资（扣15%税）
tax = total * 0.15
net_salary = total - tax

print(f"基本工资：{base_salary}元")
print(f"加班费：{overtime_pay}元")
print(f"税前总工资：{total}元")
print(f"扣税：{tax}元")
print(f"税后工资：{net_salary}元")

运行结果：

基本工资：8000元
加班费：500元
税前总工资：8500元
扣税：1275.0元
税后工资：7225.0元
