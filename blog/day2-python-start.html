¶ 👋 Day2 - 装好Python，写出第一个"Hello World"

¶ 📌 今日目标

  • 安装 Python 环境
  • 选对编辑器
  • 写出第一行代码

¶ 🔧 环境选择

¶ 我的选择

工具用途推荐理由
AnacondaPython 发行版预装常用库，不用手动折腾
VS Code代码编辑器免费、轻量、插件丰富
Git版本控制记录学习轨迹

¶ 为什么不用纯 Python？

Anaconda 预装了：

  • NumPy、Pandas（数据分析）
  • Jupyter Notebook（交互式编程）
  • 虚拟环境管理（避免版本冲突）

省心 > 省空间

¶ ⚠️ 踩坑记录

¶ 坑1：版本选择

问题：Python 2 和 Python 3 区别很大 解决：无脑选 Python 3.11+（最新稳定版）

bash
# 终端检查
python --version
python3 --version

¶ 坑2：环境变量

问题：安装后命令行找不到 python 解决：勾选 "Add Python to PATH"

或者手动配置：

bash
# Windows PowerShell
[Environment]::SetEnvironmentVariable("Path", $env:Path + ";C:\Python311", "User")

¶ 坑3：编辑器选择

问题：Notepad++ 太简陋，PyCharm 太重 解决：VS Code + Python 插件

插件推荐：

  • Python（微软官方）
  • Pylance（代码诊断）
  • Jupyter（Notebook支持）
